# mern-context-app

## 1. Iniciar servicio de mongodb:

Linea de comandos

```
  > mongod
```

## 2. Iniciar Server - Backend

Linea de comandos - ruta del backend
```
  > npm run dev
```

## 3. Iniciar cliente - Frontend

Linea de comandos - ruta del frontend
```
  > npm start
```
